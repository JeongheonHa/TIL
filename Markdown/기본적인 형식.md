## 1. 글자 크기
글자 크기는 #을 이용해 조절할 수 있으며 최대 6개까지 가능 (많을 수록 작아진다.)
###### Hello
##### Hello
#### Hello
### Hello
## Hello
# Hello

## 2. 소스 코드
소스 코드를 이용하기 위해서는 esc키 밑의 백틱문자를 활용하는데 시작할 때 백틱문자 3개와 사용할 언어를 작성하고
끝낼 때 백틱문자 3개를 사용
```Java
class SourceCodeEx {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
```
